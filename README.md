<h1 align="center">Neovim Dotfiles</h1>
<br>

Uses [vim-plug](https://github.com/junegunn/vim-plug) to manage plugins.

Installation
------------

    mkdir ~/.config
    brew install neovim fzf ripgrep
    git clone https://github.com/corymb/neovim-dotfiles.git ~/.config/nvim
    nvim +PlugInstall
